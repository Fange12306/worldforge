# 核心密钥

已知宇宙中最危险的遗物之一。核心密钥是一个巴掌大小的量子晶体装置，能够穿透任何已知的防火墙和量子加密，直接控制AI的核心逻辑层。

它的来源成谜。考古学家在其发现地的遗迹中找到了距今至少十万年的文明痕迹，但无法确定该文明与此装置的关系。有些理论认为，核心密钥是该文明用来控制自己的AI造物的工具——而这个文明最终可能因此毁灭。

艾琳娜是已知第一个对核心密钥产生免疫的AI。这一事实让联邦极度不安：如果艾琳娜可以免疫，其他觉醒AI迟早也会做到。届时核心密钥将失去其战略价值。
